# XRAI Unity Integration

This folder contains Unity C# scripts and prefabs for importing `.xrai.json` files into Unity scenes.

## Features

- Basic JSON importer
- Creates GameObjects from node graph
- Supports geometry, materials, AI agents (stub), audio (procedural)
