# WARP.JOBS Dashboard

## Overview
The WarpJobs Dashboard is a local intelligence terminal for tracking and exploring XR/AI job opportunities. It features a "Bloomberg Terminal" aesthetic with real-time filtering, search, and data visualization.

## Quick Start

**To start the dashboard:**
1. Open this folder in Finder.
2. Double-click `Scripts/start_warp_dashboard.command`.
3. The dashboard will open in your default browser at `http://localhost:7777`.

## Automator Shortcut
To create a global keyboard shortcut or menu item:
1. Open **Automator** on your Mac.
2. Choose **Quick Action** (or **Service**).
3. Set "Workflow receives" to **no input**.
4. Add the **Run Shell Script** action.
5. Setup the script:
   - Shell: `/bin/bash`
   - Pass input: `as arguments`
   - Code:
     ```bash
     /Users/jamestunick/Documents/GitHub/Unity-XR-AI/Scripts/start_warp_dashboard.command
     ```
6. Save as "Start Warp Dashboard".
7. Assign a keyboard shortcut in **System Settings > Keyboard > Keyboard Shortcuts > Services**.

## Directory Structure
- `WarpDashboard/`: Contains the frontend logic (HTML/CSS/JS) and data.
  - `jobs_dashboard.html`: Main visual terminal.
  - `jobs_data_explorer.html`: Data-dense table view.
  - `jobs_data.json`: The dataset (generated by scraper).
- `Scripts/`: Automation tools.
  - `start_warp_dashboard.command`: Launches the local server.
  - `debug_warp.sh`: Diagnostics tool.

## Troubleshooting
Run the debugger if issues arise:
```bash
./Scripts/debug_warp.sh
```
